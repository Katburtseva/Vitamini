# 💊 Vitamin Reminder Bot

Telegram-бот для напоминания о приёме витаминов.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
export BOT_TOKEN="токен_от_BotFather"
python bot.py
```

На Windows:
```cmd
set BOT_TOKEN=токен_от_BotFather
python bot.py
```

## Получить токен

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Следуй инструкциям, получи токен
4. Вставь токен в переменную окружения `BOT_TOKEN`

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/add` | Добавить витамин |
| `/list` | Список витаминов |
| `/stats` | Статистика |
| `/delete` | Удалить витамин |

## Формат времени

- `21:00` или `21.00` — точное время
- `8` — 8:00
- `утром` — 8:00
- `обед` — 13:00
- `вечером` — 20:00
- `ночью` — 22:00

## Данные

Все данные хранятся в файле `data.json` рядом с ботом.
